# AutoBay 平台集成指南

## 概述

AutoBay 通过 `MarketplaceConnector` Protocol 统一抽象了所有电商平台 API。目前已实现 4 个连接器：

| 平台 | 连接器 | API 类型 | 区域 | 状态 |
|------|--------|---------|------|------|
| eBay | EbayConnector | REST (Inventory + Fulfillment API) | AU, US, UK | 已实现 |
| Amazon | AmazonConnector | REST (SP-API) | US, AU, UK | 已实现 |
| Shopify | ShopifyConnector | GraphQL Admin API (2025-04) | Global | 已实现 |
| TikTok Shop | TikTokConnector | REST (Open API v202309) | AU, US, UK | 已实现 |

---

## 1. eBay 集成

### 获取 API 凭证

1. 注册 eBay Developer Program: https://developer.ebay.com
2. 创建 Application → 获取 `App ID`, `Dev ID`, `Cert ID`
3. 在 Application Settings 中配置 Redirect URI
4. 通过 OAuth2 Authorization Code flow 获取 User Token

### 配置

在 `.env` 或 Platform Connection 中设置：
```
EBAY_APP_ID=your_app_id
EBAY_DEV_ID=your_dev_id
EBAY_CERT_ID=your_cert_id
EBAY_REDIRECT_URI=https://your-domain.com/auth/ebay/callback
EBAY_SANDBOX=true    # 开发阶段用 sandbox
```

### 使用的 eBay API

| API | 用途 | 端点 |
|-----|------|------|
| Inventory API | 创建/更新产品和库存 | `/sell/inventory/v1/inventory_item` |
| Offer API | 创建/发布报价 | `/sell/inventory/v1/offer` |
| Fulfillment API | 订单同步和发货 | `/sell/fulfillment/v1/order` |
| Marketing API | Promoted Listings | Phase 4 |

### Marketplace ID 映射

| 区域 | Marketplace ID |
|------|---------------|
| AU | EBAY_AU |
| US | EBAY_US |
| UK | EBAY_GB |

### 速率限制

- Inventory API: ~5,000 calls/day per app
- Fulfillment API: ~5,000 calls/day
- 系统自动使用指数退避重试 429 错误

## 2. Amazon 集成

### 获取 API 凭证

1. 注册 Amazon Selling Partner Developer: https://developer-docs.amazon.com/sp-api
2. 注册 SP-API Application
3. 使用 LWA (Login with Amazon) OAuth2 获取 Refresh Token
4. 配置 IAM Role ARN（用于请求签名）

### 配置

```
AMAZON_SP_CLIENT_ID=amzn1.application-oa2-client.xxx
AMAZON_SP_CLIENT_SECRET=xxx
AMAZON_SP_REFRESH_TOKEN=Atzr|xxx
AMAZON_SP_ROLE_ARN=arn:aws:iam::xxx:role/xxx    # 可选
```

### 使用的 Amazon API

| API | 用途 | 版本 |
|-----|------|------|
| Listings Items API | 创建/更新 Listing | 2021-08-01 |
| Orders API | 订单同步 | v0 |
| Feeds API | 批量操作、发货确认 | 2021-06-30 |
| Product Pricing API | 竞品价格 | Phase 3 |
| Catalog Items API | 产品搜索 | Phase 2 |

### Marketplace ID 映射

| 区域 | Marketplace ID | API 端点 |
|------|---------------|---------|
| US | ATVPDKIKX0DER | sellingpartnerapi-na.amazon.com |

## 3. Shopify 集成

### 获取 API 凭证

1. 在 Shopify Partner Dashboard 创建 App
2. 或在 Shopify Admin → Settings → Apps → Develop apps
3. 获取 Admin API Access Token
4. 记下 Store URL (your-store.myshopify.com)

### 配置

```
SHOPIFY_API_KEY=your_api_key
SHOPIFY_API_SECRET=your_secret
SHOPIFY_STORE_URL=your-store.myshopify.com
```

Platform Connection credentials:
```json
{
  "store_url": "your-store.myshopify.com",
  "access_token": "shpat_xxx",
  "location_id": "gid://shopify/Location/12345"  // 用于库存管理
}
```

### 重要说明

- **必须使用 GraphQL Admin API**（REST API 已在 2025.4 废弃）
- API 版本：`2025-04`
- 速率限制：50 cost points/second（GraphQL）

### 使用的 GraphQL Mutations/Queries

| 操作 | GraphQL | 用途 |
|------|---------|------|
| productCreate | Mutation | 创建产品 |
| productUpdate | Mutation | 更新产品 |
| productDelete | Mutation | 删除产品 |
| productVariantUpdate | Mutation | 更新价格 |
| inventoryAdjustQuantities | Mutation | 调整库存 |
| orders (query) | Query | 查询订单 |
| fulfillmentCreateV2 | Mutation | 发货 |

## 4. TikTok Shop 集成

### 获取 API 凭证

1. 注册 TikTok Shop Partner Center: https://partner.tiktokshop.com
2. 创建 App → 获取 App Key + App Secret
3. 通过 OAuth2 获取 Access Token + Shop Cipher

### 配置

```
TIKTOK_APP_KEY=your_app_key
TIKTOK_APP_SECRET=your_app_secret
```

Platform Connection credentials:
```json
{
  "app_key": "xxx",
  "app_secret": "xxx",
  "access_token": "xxx",
  "refresh_token": "xxx",
  "shop_cipher": "xxx"
}
```

### API 签名

TikTok Shop API 使用 HMAC-SHA256 签名。系统自动处理：
1. 按字母序排列参数
2. 拼接 path + 参数键值对
3. 使用 app_secret 作为 HMAC key
4. 签名附加到请求参数

### 使用的 API

| API | 用途 | 版本 |
|-----|------|------|
| Products API | 创建/更新产品 | v202309 |
| Orders API | 订单搜索和管理 | v202309 |
| Fulfillment API | 发货 | v202309 |

## 5. 添加新平台

要添加一个新的电商平台（如 Harvey Norman），只需：

### Step 1: 创建连接器文件

```
backend/app/integrations/harvey_norman/
├── __init__.py
└── client.py
```

### Step 2: 实现 MarketplaceConnector 接口

```python
from app.integrations.registry import ConnectorRegistry
from app.modules.products.models import Platform

@ConnectorRegistry.register(Platform.HARVEY_NORMAN)
class HarveyNormanConnector:
    def __init__(self, credentials: dict, region: str):
        ...

    async def create_listing(self, data): ...
    async def fetch_orders(self, since): ...
    # ... 实现所有接口方法
```

### Step 3: 完成

不需要修改任何业务代码。ConnectorRegistry 自动发现新连接器。
