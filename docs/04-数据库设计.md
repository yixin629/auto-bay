# AutoBay 数据库设计

## 概述

- 数据库：PostgreSQL 16
- ORM：SQLAlchemy 2.0+ (async)
- 迁移工具：Alembic
- 共计 **16 张表**
- 所有主键使用 UUID v4
- 时间字段统一使用 `TIMESTAMPTZ`（带时区）

---

## 通用 Mixin

```python
class UUIDMixin:
    id: UUID  # 主键，默认 uuid4()

class TimestampMixin:
    created_at: DateTime(timezone=True)  # 自动设置
    updated_at: DateTime(timezone=True)  # 自动更新
```

### 9-11. inventory_locations / inventory_items / inventory_movements — 库存三表

- **locations**: 仓库/3PL/供应商/虚拟库位
- **items**: 每个产品在每个库位的库存量（on_hand, reserved）
- **movements**: 入库/出库/调整/退货/调拨的流水记录

### 12-13. exchange_rates / price_history — 汇率 + 价格历史

- **exchange_rates**: 时序表，每次刷新插入新行
- **price_history**: 记录每次价格变动（旧价→新价+原因）
