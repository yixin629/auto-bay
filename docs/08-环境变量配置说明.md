# AutoBay 环境变量配置说明

所有配置通过 `.env` 文件管理（基于 `pydantic-settings` 自动加载）。

---

## 必须配置

| 变量 | 示例值 | 说明 |
|------|--------|------|
| `SECRET_KEY` | `aB3dEf...` (32+ chars) | 应用加密密钥，`python -c "import secrets; print(secrets.token_urlsafe(32))"` |
| `JWT_SECRET_KEY` | `xY7zW...` (32+ chars) | JWT 签名密钥，同上方法生成 |
| `DATABASE_URL` | `postgresql+asyncpg://autobay:autobay@localhost:5432/autobay` | PostgreSQL 连接串 |

---

## AI 功能（至少配一个）

| 变量 | 说明 |
|------|------|
| `ANTHROPIC_API_KEY` | Anthropic Claude API Key，从 console.anthropic.com 获取 |
| `OPENAI_API_KEY` | OpenAI API Key（可选备用），从 platform.openai.com 获取 |

---

## 平台凭证

### eBay

| 变量 | 说明 |
|------|------|
| `EBAY_APP_ID` | eBay Developer Program 的 Application ID |
| `EBAY_DEV_ID` | Developer ID |
| `EBAY_CERT_ID` | Certificate ID |
| `EBAY_REDIRECT_URI` | OAuth 回调 URL |
| `EBAY_SANDBOX` | `true` 用 sandbox 环境，`false` 用生产环境 |

### Amazon SP-API

| 变量 | 说明 |
|------|------|
| `AMAZON_SP_CLIENT_ID` | LWA Client ID (amzn1.application-oa2-client.xxx) |
| `AMAZON_SP_CLIENT_SECRET` | LWA Client Secret |
| `AMAZON_SP_REFRESH_TOKEN` | Seller 授权后获取的 Refresh Token |
| `AMAZON_SP_ROLE_ARN` | IAM Role ARN（用于请求签名，可选） |

### Shopify

| 变量 | 说明 |
|------|------|
| `SHOPIFY_API_KEY` | Shopify App API Key |
| `SHOPIFY_API_SECRET` | Shopify App Secret |
| `SHOPIFY_STORE_URL` | 你的 Shopify 店铺域名 (xxx.myshopify.com) |

### TikTok Shop

| 变量 | 说明 |
|------|------|
| `TIKTOK_APP_KEY` | TikTok Shop Partner Center App Key |
| `TIKTOK_APP_SECRET` | App Secret |

---

## 基础设施

| 变量 | 默认值 | 说明 |
|------|--------|------|
| `REDIS_URL` | `redis://localhost:6379/0` | Redis 连接 |
| `CELERY_BROKER_URL` | `redis://localhost:6379/1` | Celery 消息队列 |
| `CELERY_RESULT_BACKEND` | `redis://localhost:6379/2` | Celery 结果存储 |
| `S3_ENDPOINT` | `http://localhost:9000` | MinIO/S3 端点 |
| `S3_ACCESS_KEY` | `minioadmin` | MinIO 访问密钥 |
| `S3_SECRET_KEY` | `minioadmin` | MinIO 秘密密钥 |
| `S3_BUCKET` | `autobay` | 存储桶名称 |
| `S3_REGION` | `us-east-1` | 区域 |

---

## 应用配置

| 变量 | 默认值 | 说明 |
|------|--------|------|
| `APP_NAME` | `AutoBay` | 应用名称 |
| `APP_ENV` | `development` | 环境：development / staging / production |
| `DEBUG` | `true` | 调试模式（控制 Swagger 文档显示、SQL 日志等） |
| `API_V1_PREFIX` | `/api/v1` | API 路径前缀 |
| `JWT_ALGORITHM` | `HS256` | JWT 签名算法 |
| `JWT_ACCESS_TOKEN_EXPIRE_MINUTES` | `30` | Access Token 有效期（分钟） |
| `JWT_REFRESH_TOKEN_EXPIRE_DAYS` | `7` | Refresh Token 有效期（天） |

---

## 生产环境建议

```ini
APP_ENV=production
DEBUG=false

# 使用强密钥
SECRET_KEY=<64字符随机字符串>
JWT_SECRET_KEY=<64字符随机字符串>

# 使用 SSL 连接数据库
DATABASE_URL=postgresql+asyncpg://user:pass@db-host:5432/autobay?ssl=require

# 使用 Redis 集群
REDIS_URL=redis://:password@redis-host:6379/0

# 使用 AWS S3 替代 MinIO
S3_ENDPOINT=https://s3.ap-southeast-2.amazonaws.com
S3_ACCESS_KEY=AKIA...
S3_SECRET_KEY=xxx
S3_BUCKET=autobay-prod
S3_REGION=ap-southeast-2

# 关闭 eBay Sandbox
EBAY_SANDBOX=false
```
